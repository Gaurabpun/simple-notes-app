# Simple Notes App

A basic notes web app built using Flask. Users can add notes and view them on the page.

## Features
- Add notes using a form
- Display notes dynamically on the page
- Simple frontend using HTML, CSS, and JavaScript
- Backend built with Flask

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure

```
simple-notes-app/
├── app.py
├── README.md
├── static/
│   ├── style.css
│   └── script.js
└── templates/
    └── index.html
```

## How to Run
1. Clone the repository
2. Open the project folder
3. Install Flask if needed:
   pip install flask
4. Run the app:
   python app.py
5. Open the local server link in your browser
